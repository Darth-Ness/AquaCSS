AquaCSS

AquaCSS is a CSS framework.


It is designed to be fast, lightweight and powerful.
It is very simple to use.
All you have to do is add the AquaMain.css and a theme of your choice to your webpage.
Then, AquaCSS will do the rest for you!



If you would like to add something to AquaCSS please create a pull request on the GitHub link above.
If you have a suggestion or you found a bug please create a issue.
